Features

Six powerful sections for monitoring and managing your Claude Code usage:

📊 Overview

Real-time metrics with:

  • Today vs. all-time costs
  • Token breakdown (input/output/cache)
  • 30-day trend chart
  • Monthly projections

🗂️ Projects

Browse all projects:

  • Sessions per project
  • Total tokens used
  • Cost per project
  • Quick filtering

📝 Sessions

Full session details:

  • Message threads
  • Date filtering
  • Full-text search
  • Per-session costs

💰 Budget

Subscription & spending:

  • Plan selection
  • Budget limits
  • Pricing editor
  • Cost tracking

⭐ Data Management

Rebuild & history:

  • One-click rebuild
  • Real-time progress
  • Rebuild history table
  • Statistics tracking

⚙️ Settings

Configuration control:

  • Timezone selection
  • Model pricing
  • Live updates
  • No restart needed

Quick Start


# Build the binary
make build

# Index all existing sessions
./bin/tokenpulse index

# Start the dashboard
./bin/tokenpulse serve

# Open http://127.0.0.1:3456 in your browser

For detailed setup instructions, see README.md
