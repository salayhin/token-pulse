# Homebrew tap

This project is distributed via Homebrew through a tap repository.

## One-time setup (project owner)

1. Create a public repo named `homebrew-tap` under the same GitHub owner
   as this project (e.g. `github.com/sirajus-salayhin/homebrew-tap`).
   It can be empty.
2. Create a Personal Access Token with `repo` scope on the tap repo and
   add it as a secret named `HOMEBREW_TAP_TOKEN` on this project.
3. Tag a release: `git tag v0.1.0 && git push origin v0.1.0`.
   GoReleaser will build cross-platform binaries, publish a GitHub
   release, and commit a generated `Formula/tokenpulse.rb` to
   the tap repo.

## End-user install

```bash
brew install sirajus-salayhin/tap/tokenpulse
```

## Files in this directory

- `tokenpulse.rb.template` — reference formula shape; the real
  one is auto-generated by GoReleaser on release.
